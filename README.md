## 📖 Overview

A PHP and MySQL web application that allows students to register and store their information securely.

## ✨ Features

- Student registration
- Database storage
- Form validation
- Error handling

## 🛠 Technologies Used

- PHP
- MySQL
- HTML5
- CSS3

## 📂 Project Structure

```text
assets/
database/
screenshot/
db_config.php
index.html
process.php
```

## 🚀 Installation

1. Clone the repository
2. Import `database.sql`
3. Start Apache & MySQL in XAMPP
4. Open `index.html`
